# AI Resume Analyzer 📝💻

An AI-based web application that analyzes and ranks resumes based on job descriptions using Natural Language Processing (NLP).

## 🚀 Features

* Upload job description and resumes (PDF)
* Extract text using PyPDF2
* Perform NLP processing using spaCy
* Calculate similarity using TF-IDF and cosine similarity
* Rank resumes based on relevance
* Export ranked results as CSV

## 🛠️ Tech Stack

* Python
* Flask
* spaCy
* Scikit-learn
* HTML, CSS

## ▶️ How to Run

```bash
git clone https://github.com/sambatkumar/AI-resume-ranker.git
cd AI-resume-ranker
pip install -r requirements.txt
python app.py
```
